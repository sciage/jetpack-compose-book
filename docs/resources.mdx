## Libraries

| Project Name | Introduction |
| -------|------|
|[Accompanist](https://github.com/google/accompanist) | Some functional supplementary libraries for Jetpack Compose
|[Ucrop](https://github.com/Yalantis/uCrop) | It can complete the cropping of pictures, which is helpful for the user's picture upload function (known to be used by Bilibili)
|[Material Design](https://material.io/design) | Android application design standards, rich icon button resources
|[CMaterialColors](https://github.com/FunnySaltyFish/CMaterialColors) | Contains all MaterialDesignColors for easy use in Jetpack Compose
|[Material Dialog](https://github.com/afollestad/material-dialogs) | Contains color, time selection dialogs, etc.
|[Coil](https://coil-kt.github.io/coil/compose/) | Compose additional image library, support to display Svg Gif, support to display pictures of network resources
|[Landscapist](https://github.com/skydoves/Landscapist) | It is also a library for image loading, which supports Glide, Coil, Fresco and other libraries
|[ComposeDataSaver](https://github.com/FunnySaltyFish/ComposeDataSaver) | Elegant data persistence in Jetpack Compose
